# AeroScope – Global AQI Rankings

**AeroScope** is a lightweight, web-based dashboard designed to track and visualize Global Air Quality Index (AQI) rankings in real-time. This project provides a clean, user-friendly interface to monitor air quality across various locations, helping users stay informed about environmental conditions.

## 🚀 Features
* **Real-time AQI Rankings:** Displays up-to-date air quality data from global sources.
* **Minimalist UI:** A "pill" styled interface designed for quick data consumption.
* **Responsive Design:** Fully optimized for viewing on both desktop and mobile devices.
* **Dynamic Loading:** Uses JavaScript to fetch and render rankings into a central container.

## 🛠️ Built With
* **HTML5:** Structured for semantic clarity and accessibility.
* **CSS3:** Custom styling featuring a modern, "app-like" aesthetic.
* **JavaScript (ES6+):** Handles data fetching and dynamic content updates within the `ranking-container`.

## 📂 Project Structure

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/MonishaCM155/Aerospace.git
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd Aerospace
    ```

    Open `index.html` in any modern web browser to view the rankings.
